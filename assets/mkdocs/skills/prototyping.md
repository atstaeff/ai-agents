# Prototyping & Playtesting

Rapid Prototyping Patterns fuer Game Prototypes, Creative App MVPs und Game Jam Projekte.

## Inhalt

- **1-Hour Game Prototypes** — Phaser, PyGame, HTML5 Canvas Templates
- **30-Sekunden Regel** — Sofort testbare Core Mechanics
- **Playtesting Framework** — Session Templates, Beobachtungsmethoden
- **Automatisierte Metriken** — Death Heatmaps, Session-Analyse
- **Game Jam Strategien** — 48-Stunden Timeline, Survival Rules

## Prototype vs. Skip

| Prototyp bauen ✅ | Spaeter machen ❌ |
|-------------------|-------------------|
| Core Mechanic | Menues, Settings |
| Visual Feedback | Polished Art |
| Win/Lose State | Save System |
| Ein kompletter Loop | Multiple Levels |
