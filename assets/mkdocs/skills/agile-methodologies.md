# Agile Methodologies

> Scrum, Kanban, Sprint Planning und agile Praktiken fuer effektive Softwareentwicklung.

---

## Frameworks

### Scrum

| Element | Beschreibung |
|---------|-------------|
| Sprint | Zeitbox (1-4 Wochen) |
| Sprint Planning | Sprintziel und Backlog festlegen |
| Daily Standup | 15 Min: Was getan, was geplant, Blocker |
| Sprint Review | Ergebnisse demonstrieren |
| Retrospektive | Prozessverbesserung reflektieren |

### Kanban

| Prinzip | Beschreibung |
|---------|-------------|
| Visualize Work | Board mit Spalten fuer Workflow |
| WIP Limits | Maximal parallele Arbeit begrenzen |
| Flow Management | Durchlaufzeit optimieren |
| Continuous Delivery | Kein fester Sprint-Zyklus |
