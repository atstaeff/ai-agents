# Game Design Principles

Game Design Prinzipien fuer das Entwerfen von Core Loops, Progression Systems, Difficulty Curves und Player Experiences.

## Inhalt

- **Core Loop Design** — Action → Reward → Decision Zyklen
- **Game Design Document (GDD) Template** — Strukturierte Planungsvorlage
- **Difficulty Curves** — Linear, Stepped, Wave, Dynamic
- **Progression Systems** — Power, Knowledge, Content, Social, Mastery
- **Reward Scheduling** — Fixed/Variable Ratio & Interval
- **Player Psychology** — Bartle's Types, Self-Determination Theory
- **Balancing** — Resource Economy, Playtesting Questions

## Core Loop Beispiele

| Genre | Action | Reward | Decision |
|-------|--------|--------|----------|
| Platformer | Jump, Run, Dodge | Coins, Neue Area | Welcher Pfad, Wann riskieren |
| RPG | Fight, Explore, Talk | XP, Items, Story | Build, Party, Quests |
| Puzzle | Match, Rotate, Place | Score, Neue Levels | Welche Zuege, Reihenfolge |
| Roguelike | Explore, Fight, Die | Unlocks, Knowledge | Risk vs. Safety, Builds |
