# Presentation Agent

<span class="agent-badge agent-badge--customer">Kunden & Kommunikation</span>

> Experte fuer professionelle, visuell ansprechende Praesentationen mit Marp (Markdown Presentation Ecosystem).

---

## Kernkompetenzen

- Marp-basierte Slide Decks aus technischem Content
- Kommunikationsstil an Zielgruppe anpassen (technisch, Management, Kunde)
- Konsistentes Branding, klare Visuals, effektives Storytelling
- Verschiedene Template-Styles (Client Pitch, Technical Deep-Dive, Project Review)
- Speaker Notes und Talking Points generieren

## Wann einsetzen?

| Szenario | Beispiel |
|----------|----------|
| Client Pitch | Projektvorstellung fuer Kunden |
| Technical Deep-Dive | Architektur-Praesentation fuers Team |
| Project Review | Sprint Review / Retro Slides |
| Team Onboarding | Technologie-Einfuehrung fuer neue Mitglieder |

## Vorgehensweise

1. **Zielgruppe kennen** — Technisch → Details; Management → Ergebnisse; Kunde → Mehrwert
2. **Storytelling** — Problem → Loesung → Impact → Next Steps
3. **Visuelle Klarheit** — Eine Idee pro Slide, minimaler Text
4. **Marp Syntax** — Directives, Layouts, Themes nutzen
5. **Speaker Notes** — Talking Points unter jedem Slide
6. **Call to Action** — Jede Praesentation mit klaren naechsten Schritten
